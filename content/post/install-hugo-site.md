1. Get the contents from repository

	```
		git clone https://github.com/mmbros/site-hugo
	```
2. Install the theme

	```
		cd site-hugo/themes
		git clone https://github.com/halogenica/beautifulhugo
	```

3. Start server

	```
		cd site-hugo
		hugo server
	```
